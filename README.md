# Install

```
mkdir project
cd project
git clone git@github.com:entoj/entoj-blueprint-sass-jspm.git .
npm install
```

# Commands

## Start server
```npm start```

## Install javascript dependency via jspm
```npm run jspm install jquery```

## Get a list if all commands
```./entoj.sh```

## Add a new entity
```./entoj.sh scaffold entity a-example```
